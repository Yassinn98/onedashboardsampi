#OneDashboard

A comprehensive analytics dashboard for mobile app monitoring with crash reporting, session replay, in-app ratings, and bug tracking.

## Features

- **App Summary**: KPIs, charts, and analytics overview
- **Crash Reporting**: Real-time crash monitoring with stack traces
- **Session Replay**: User session recordings with rage click detection
- **In-App Ratings**: Star ratings and review management
- **Reported Bugs**: Jira-integrated bug tracking

## Pages

- `dashboard.html` - Main analytics dashboard
- `crash-reporting.html` - Crash monitoring and analysis
- `session-replay.html` - Session recordings
- `in-app-ratings.html` - User ratings and reviews
- `reported-bugs.html` - Bug tracking with Jira integration

## Technologies

- HTML5, CSS3, JavaScript
- Chart.js for data visualization
- jsPDF & html2canvas for PDF export
- Responsive design with mobile support
